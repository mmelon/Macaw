How to use px and % at the same time? Use % for sizing your element and use margin or padding for integrating a px sized object inside of it. create a container with a left padding the size of your left element. In this my example file you'll see: |--div <-- size in px or % with px left padding | |--img <-- absolute position size in px | |--content <-- 100% width | | |--h1 | | |--p
