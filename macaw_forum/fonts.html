HACKS #2 Using web fonts within Macaw 1. from your favorites type foundries. Example: http://ngenty.com/macaw/webfonts/manual/ - file Note: zip includes the web font folder (Outside the published folder because Macaw rewrites it when publishing). Font used: Clear Sans 1.00 (download it here). 2. from google fonts (and not already installed in Macaw) Example: http://ngenty.com/macaw/webfonts/google/ - file Trick here is to change the font-family name because Macaw use the full name (family + weight) Font used: Pacifico (you can download it here). 3. from an opentype file (.otf or .ttf). Use the webfont generator from Fontsquirrel: http://www.fontsquirrel.com/tools/webfont-generator. Note: Read the agreement because fonts are subjects to copyright, so be sure to have the right to use them on your website. If not, buy a proper license. After that, use them like in my first example. Note: If you see this in Macaw: Macaw requires that you install the typo in your computer before using it. To make it simple, Macaw needs a system font and your users needs a web font (yeah it's sounds dumb, i know). All you have to do is using the system font inside Macaw, then manually attach the web font to your page (inside your “head & tail”), like this: Tips: after using the system font with Macaw, publish your work and preview your css, Macaw uses sometimes the full font name for the font-family (font-family + weight). This is this name you need to use in your head & tail code.
